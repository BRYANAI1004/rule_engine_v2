## Admission

- Admission is indicated for 1 or more of the following ( 1 ) ( 2 ) ( 3 ) ( 4 ):
  - Hemodynamic instability ( 1 ) ( 2 ) ( 3 ) ( 7 ) ( 8 ) ( 9 )
  - Severe electrolyte abnormalities requiring inpatient care ( 3 ) ( 7 )
  - Cardiac arrhythmias of immediate concern ( 3 ) ( 7 )
  - Acute myocardial ischemia causing or associated with failure. See Angina ISC or Myocardial Infarction ISC guideline as appropriate. ( 1 ) ( 3 ) ( 7 )
  - Pulmonary edema that is very severe (eg, invasive or noninvasive assisted ventilation needed, [ A ] imminent, or likely; or need for 100% oxygen to keep oxygen saturation above 90%) ( 3 ) ( 10 )
  - Anasarca or peripheral edema that is severe (eg, significant paroxysmal dyspnea, inability to ambulate or void, skin breakdown, concomitant ascites) ( 2 ) ( 11 )
  - Pulmonary edema and ALL of the following ( 3 ) ( 7 ):
    - New need for oxygen therapy to keep oxygen saturation above 90% (or increased FiO 2 need from baseline)
    - Has not improved sufficiently with observation care (eg, appropriately dosed IV diuretics) [ B ]
  - Tachypnea that persists despite observation care ( 7 )
  - Dyspnea (above baseline) that persists despite observation care ( 7 )
  - Altered mental status that is severe or persistent ( 8 )
  - Increased creatinine with reduction of more than 50% in estimated glomerular filtration rate from baseline [ C ] eGFR - Adult Calculator ( 3 )
  - Progressively (ongoing) rising creatinine with reduction of more than 25% in estimated glomerular filtration rate from baseline [ C ] eGFR - Adult Calculator ( 3 ) ( 7 )
  - Pulmonary artery catheter monitoring needed [ D ] ( 1 ) ( 8 )
- Alternatives include ( 1 ) ( 2 ) ( 3 ) ( 17 ) ( 18 ) ( 19 ) ( 20 ):
  - For arrhythmia: emergency department or rapid treatment site intervention ( 21 )
    - Pharmacologic therapy to slow supraventricular arrhythmias
    - Cardioversion (electrical or pharmacologic) to sinus rhythm
  - For heart failure with pulmonary congestion or peripheral edema:
    - Outpatient therapy (eg, renin-angiotensin inhibitor (angiotensin receptor-neprilysin inhibitor preferred), diuretics, beta-blocker, aldosterone antagonist, sodium-glucose cotransporter-2 inhibitor (eg, dapagliflozin, empagliflozin), vericiguat, or vasodilator (hydralazine, isosorbide), patient education, nutritional and lifestyle interventions for malnutrition or obesity) [ B ] [ E ] [ F ] [ G ] ( 1 ) ( 4 ) ( 18 ) ( 25 ) ( 26 )
    - Multidisciplinary team care with home care support ( 1 ) ( 27 )
    - Emergency department or rapid treatment site care with oxygen, respiratory therapy, IV diuretics, correction of fluid and electrolyte imbalance, and control of arrhythmias ( 17 ) ( 28 )
    - Recovery facility care for ongoing parenteral medication, oxygen therapy, weight and fluid balance, laboratory testing, electrolyte correction, and management of comorbidities
    - Skilled nursing facility for medication adjustment, weight and fluid balance, nursing assessment, and nutritional support ( 29 )
  - Observation. See Heart Failure: Observation Care ISC guideline as appropriate.
  - Palliative care ( 4 ) ( 30 )
- Extended stay beyond goal length of stay may be needed for ( 1 ) ( 2 ) ( 6 ) ( 36 ):
  - Failure to meet discharge criteria (recovery milestones within final day in Optimal Recovery Course)
    - Expect brief stay extension.
  - Acute cardiac ischemia, confirmed or suspected as precipitant or consequence of heart failure ( 32 ) ( 37 )
    - Cardiac ischemia may prompt rapid cardiac catheterization and revascularization with angioplasty, CABG, or thrombolytic therapy.
    - Anticipate possible temporary perioperative mechanical cardiac support (eg, intra-aortic balloon pump).
    - Anticipate intervention within 24 hours.
    - Expect brief stay extension.
  - Cardiogenic shock ( 8 ) ( 9 ) ( 31 ) ( 32 ) ( 38 )
    - Anticipate rapid supportive therapy and intervention for treatable causes.
    - Pulmonary artery catheter, inotropic agents and vasopressor agents, CPAP or noninvasive ventilation, [ A ] intra-aortic balloon pump, ventricular assist device, or cardiac catheterization may be used as indicated. ( 9 ) ( 14 ) ( 15 )
    - Intubation and mechanical ventilation may be required due to cardiogenic pulmonary edema and increased ventilation requirements.
    - Treat identifiable causes:
      - For acute valvular disease (eg, chordae tendineae rupture, aortic dissection with valve involvement, critical stenosis): rapid cardiac surgery if possible
      - For pericardial tamponade: pericardiocentesis
      - For pulmonary embolus: anticoagulation, possible thrombolytics, possible embolectomy, possible caval filter
    - Anticipate treatment of systemic complications of shock (eg, acute renal failure, thrombocytopenia, delirium, gastrointestinal bleeding, infection). ( 9 )
    - Expect brief to moderate stay extension.
    - See Hemodynamic Instability: Common Complications and Conditions ISC guideline as appropriate.
  - Acute kidney injury (stage 2) or Acute renal failure (stage 3 acute kidney injury) ( 32 )
    - Anticipate possible renal replacement therapy to control fluid overload if diuresis is not sufficient.
    - Acute kidney injury may occur as a result of cardiogenic shock.
    - Expect brief stay extension.
  - Stage IV chronic kidney disease (estimated glomerular filtration rate of less than 30 mL/min/1.73m 2 (0.50 mL/sec/1.73m 2 ), and not previously on chronic dialysis) eGFR - Adult Calculator ( 39 ) ( 40 ) ( 41 )
    - Anticipate need for more intensive diuretic therapy (higher dose or second diuretic) and electrolyte monitoring.
    - Ultrafiltration or renal replacement therapy may be required for diuretic resistance or failure.
    - Expect brief stay extension.
  - Respiratory failure (eg, need for noninvasive [ A ] or invasive assisted ventilation) ( 4 ) ( 32 ) ( 42 )
    - Assisted ventilation [ A ] may be required due to cardiogenic pulmonary edema or increased ventilation requirements due to acidosis or shock.
    - Anticipate aggressive diuresis, respiratory support, and frequent monitoring of clinical status.
    - Expect brief to moderate stay extension.
    - See Respiratory Insufficiency: Common Complications and Conditions ISC guideline as appropriate.
  - Concomitant pneumonia
    - Anticipate clinical testing (eg, imaging, blood tests) and treatment (eg, antibiotics).
    - Expect brief stay extension.
  - Significant electrolyte abnormality (severe hyponatremia, hypokalemia)
    - Anticipate fluid restriction, and salt or electrolyte repletion.
    - Expect brief stay extension.
  - Newly diagnosed (new-onset) atrial fibrillation ( 7 ) ( 21 )
    - Anticipate medication to control heart rate and possible need for cardioversion (electrical or pharmacologic).
    - Patient may need initiation of anticoagulation. See Anticoagulation Requirement: Common Complications and Conditions ISC guideline.
    - Expect brief stay extension.
  - Acute ventricular arrhythmia ( 43 )
    - Anticipate correction of reversible causes (eg, electrolyte repletion, acidosis, discontinuation of proarrhythmic agents).
    - Anticipate antiarrhythmic therapy and possible overdrive pacing, ablation, ICD placement or ICD reprogramming.
    - Expect brief stay extension.
  - Acute valvular heart disease ( 32 )
    - Anticipate possible percutaneous valvuloplasty, transcatheter valve replacement, or urgent surgical intervention.
    - Expect brief to moderate stay extension.
  - Amyloid-related heart failure ( 44 )
    - Amyloid-related heart failure is associated with a higher need for vasopressor agents and higher mortality.
    - Expect brief stay extension.
